# lab One
my terraform labs
lab Requirements :
 - terraform installed
 - openstack CLI installed
 - a non-interactive version of your tenant's openrc

## How to
Read and use terraform.tfvars.sample to set variables :
 - cp terraform.tfvars.sample terraform.tfvars
 - set variables as commented

To launch terraform
 - terraform init
 - terraform plan
 - terraform apply
